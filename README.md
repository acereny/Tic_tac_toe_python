# Simple Two-Player Tic-Tac-Toe

A classic Tic-Tac-Toe game built with Python for two players on the same machine.

## How to Play
1. Run the script: `python tic_tac_toe.py`
2. Players take turns entering numbers from **1 to 9**.
3. The first player to get 3 marks in a row (horizontal, vertical, or diagonal) wins!
